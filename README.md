# 🌐 FreshBarber — Лендинг под ключ

Адаптивный одностраничный сайт для мужской барбершоп-студии. Разработан с нуля без фреймворков — чистый HTML/CSS/JS.

## Стек

- HTML5 + семантическая разметка
- CSS3 (переменные, grid, flexbox, анимации)
- Vanilla JavaScript (без зависимостей)
- Google Fonts (Inter)

## Страницы / Секции

| Секция | Описание |
|--------|----------|
| Hero | Полноэкранный блок с CTA и статистикой |
| Услуги | 4 карточки с иконками и описанием |
| Мастера | Сетка с аватарами, специализацией и рейтингом |
| Цены | Таблица услуг с ценами |
| Запись | Форма записи с маской телефона |
| Контакты | Адрес, телефон, часы работы |

## Особенности

- ✅ Полностью адаптивный (mobile-first)
- ✅ Sticky-шапка с эффектом blur
- ✅ Плавный scroll и IntersectionObserver-анимации
- ✅ Маска телефонного номера
- ✅ Hamburger-меню на мобильных
- ✅ Форма записи с обратной связью
- ✅ SEO-метатеги (title, description, viewport)

## Запуск

Просто открой `index.html` в браузере или подними любой статический сервер:

```bash
npx serve .
# или
python -m http.server 8080
```

## Скриншоты

Дизайн использует тёмную гамму (midnight dark) с фиолетовым акцентом (#6A5ACD) и голубым хайлайтом (#00CFFF).
